## 🚀 Featured projects

### 📊 [FlowDesk](https://github.com/HugoJunioor/FlowDesk)
Slack-based demand management system with **business-hours SLA**, executive dashboards
and AI-driven auto-classification.

🌐 **Demo:** [flow-desk-e2is.vercel.app](https://flow-desk-e2is.vercel.app)
🔑 Login: `master` / Password: `Admin@1`

**Highlights:**
- React 18 + TypeScript + Vite
- 8 modules × 5 actions of granular permissions
- Interactive BI reports (auto-generated HTML) + Excel export
- Multi-channel adapters (Slack ✅ Teams 📋 Discord ⏳)
- Full hardening: PBKDF2, CSP, HSTS, rate-limit, branch protection
- 100+ unit tests, Storybook, Lighthouse CI, bundle-size checks
